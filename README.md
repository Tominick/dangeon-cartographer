# Dungeon Cartographer ⚔️

A web-based dungeon mapping tool for tabletop RPGs. Design maps, place player tokens, and manage game state.

## Features
- Interactive tile-based map editor
- Multiple terrain types (grass, walls, water, doors, etc.)
- Player token management
- Save/load maps and game states
- Random dungeon generation

## Try it
You can play with it: <br/>
[https://www.micro-progs.com/dc/](https://www.micro-progs.com/dc/)

## Usage
Open `index.html` in a web browser to start.
